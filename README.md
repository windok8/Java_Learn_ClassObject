# Java_Learn_ClassObject

## Class_Object_01
### 项目介绍
```markdown
这是一个菜谱信息管理系统
    主要功能有：
        1.添加菜谱
        2.删除菜谱
        3.修改菜谱
        4.查看菜谱
        5.评论菜谱
    用户级别对应的权限如下：
        1.超级管理员：拥有所有权限
        2.管理员：拥有审核菜谱的权限
        3.作家：拥有添加、删除、修改、查看菜谱的权限，拥有评论菜谱的权限
        5.游客：拥有查看菜谱的权限
```
